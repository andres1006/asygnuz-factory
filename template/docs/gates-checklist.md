# Gates Checklist por Fase

## Gate 1 - Producto
- [ ] PRD completo
- [ ] RF completos
- [ ] RNF completos

## Gate 2 - Diseño
- [ ] Flujos de usuario documentados
- [ ] Wireframes base disponibles

## Gate 3 - Arquitectura/DB
- [ ] Arquitectura aprobada
- [ ] Modelo DB definido (DB-first)
- [ ] ADRs clave registrados

## Gate 4 - Desarrollo
- [ ] HU implementadas según criterios
- [ ] PRs con evidencia

## Gate 5 - QA/UAT
- [ ] Tests completos
- [ ] Cobertura >= 90%
- [ ] UAT ejecutado

## Gate 6 - Release
- [ ] Checklist de seguridad
- [ ] Deploy QA -> Producción validado
- [ ] Matriz de trazabilidad actualizada
