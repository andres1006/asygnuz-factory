# Factory Governance

## Objetivo
Asegurar entregas semanales con calidad, trazabilidad y consistencia entre proyectos.

## Gates globales
1. Producto completo (PRD + RF + RNF)
2. Diseño y flujos validados
3. Arquitectura/DB-first aprobados
4. Desarrollo con PRs trazables
5. QA/UAT aprobados
6. Seguridad mínima aprobada
7. Release controlado

## Definición de Hecho (DoD)
- Requisito trazado de PRD a HU, código, test y UAT
- Cobertura y pruebas dentro del estándar
- Checklist de seguridad ejecutado
