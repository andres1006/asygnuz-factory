# Quality Metrics Framework

## Métricas de entrega
- Throughput semanal
- Cycle time por HU
- % objetivos semanales logrados

## Métricas de calidad
- Cobertura de tests
- Defectos escapados a UAT
- Incidentes post-release

## Métricas de seguridad
- Hallazgos críticos/altos abiertos
- Tiempo de remediación
