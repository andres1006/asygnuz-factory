# Roles and Handoffs

## Roles
- Producto
- Diseño UX/UI
- Arquitectura
- DB Expert
- Backend
- Frontend
- QA Automatización/Integración
- UAT (Negocio)
- DevOps
- Security

## Handoffs
Producto -> Diseño -> Arquitectura/DB -> Dev -> QA -> UAT -> DevOps -> Security -> Release

Cada handoff debe incluir evidencia documental en template del proyecto.
