# Weekly Runbook

## Lunes - Plan
- Definir objetivo de valor semanal por proyecto
- Revisar riesgos y bloqueos
- Confirmar HU comprometidas

## Martes/Miércoles - Build
- Desarrollo + QA continuo
- Revisión de cobertura y deuda

## Jueves - Stabilize
- Congelar scope
- Ejecutar suite completa + hardening

## Viernes - Validate & Learn
- UAT con negocio
- Deploy si aplica
- Retro: qué mejora va al template
